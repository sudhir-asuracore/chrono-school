# ChronoSchool Frontend UI

The administrative dashboard for managing resources and visualizing generated timetables.

## Tech Stack
- **Framework:** React with TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS

## Setup
```bash
npm install
```

## Running in Development
```bash
npm run dev
```

## Building for Production
```bash
npm run build
```
